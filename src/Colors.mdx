# 🎨 Sistema de Cores (Tokens)

O **DS-OneBox** utiliza um sistema de cores semântico moderno baseado no padrão de mercado (inspirado no Shadcn UI e Radix Colors). As cores são definidas em variáveis CSS (HSL) para facilitar o suporte automático ao **Dark Mode** e permitir sobrescritas simples.

Abaixo você encontra a paleta de cores completa disponível no nosso `tailwind-preset.js`.

## Como utilizar no código?

Como todas essas cores já estão injetadas no nosso `tailwind-preset.js`, você pode utilizá-las nativamente no Tailwind usando os prefixos padrão (`bg-`, `text-`, `border-`, `ring-`):

```tsx
// Exemplo de uso de cores semânticas
<div className="bg-success/10 text-success border-success">
  Operação realizada com sucesso!
</div>

// Exemplo de uso de cores informativas
<div className="bg-info text-info-foreground p-4 rounded-md">
  Você tem 3 novas mensagens.
</div>

// Exemplo com cores de gráfico
<div className="bg-chart-1 w-full h-2 rounded-full" />
```
